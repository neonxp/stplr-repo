# Stapler репозиторий NeonXP

Репозиторий для как моего самописного софта, так и софта, который писал не я, но
я его опакетил. Почему не из стандартного пакетного менеджера? Ну, во-первых,
там не всё есть. А, во-вторых, софт из этого репозитория получается
дистрибутивонезависимым, а в перспективе даёт и повторяемость пользовательского
окружения, независимо от системного окружения.

## Установка репозитория

1. Сначала нужно установить пакетный менеджер Stapler —
   https://stplr.dev/docs/guide
2. Добавление этого репозитория в Stapler:

    ```sh
    stplr repo add neonxp.ru git://git.neonxp.ru/repo
    ```

## Установка приложений

После добавления репозитория, установка любого приложения заключается в простой
команде

```
stplr install имя_приложения
```

Например,

```
stplr install recutils
```
